` Neural Network Visualization

Neural Network Visualization

Showing activations of data point:

*blue weights are positive, yellow weights are negative

Parameters:

Layers - Controls the architecture of the network. Formatted as follows: [i, h1, ... , hn, o]

Epsilon - Boundary for the random initialization of weights: -e < w < e

Alpha - Learning rate; controls the amount a gradient can update the network weights

Batch Size - Size of data batch when batch gradient descent is performed

Updates/loop - Number of gradient descent loops performed each visualization update loop. (Processing can only update the visualization so fast, so cramming numerous GD updates in each loop makes things quicker)
